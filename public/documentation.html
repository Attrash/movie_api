Movie API

Project description This will be a web app to help user retrive data about movies. this project is built as an assignment for the careerfoundry Server-side programing achievement

Request URL Description HTTP Method Request Body Data Format Response Body Data Format
Return all movies /movies Returns a list of all movies GET none JSON of movies array
Returns movies by title /movies/:title Returns a movie by its title GET none JSON object of the specific movie associated with that title
Return information about a genre /movies/genres/:genreName Returns details about the specific genre GET none JSON object of the specific genre name and details
Return data about a director /movies/directors/:director Returns details about a specific director GET none JSON object of the specific director name and details
Create a new user /users Allow the creation of a new user POST JSON object with new user information if the creation format was correct return a status code (201)with success message. if not return a status code(400) with error message.
Update user information /users/:id Allow user to update information PUT JSON object with updated information A status code (201) that the information was updated correctly with success message. if not status code(400) with error message.
Add movie to favorites /users/:id/favorites Allow user to add a movie to their favorites list POST JSON object with movie title if the creation format was correct return a status code (201)with success message. if not return a status code(400) with error message.
Delete a movie from favorites /users/:id/favorites Allow user to delete a movie from their favorites list DELETE none A status code (201) if the movie was deleted successfully with a confirmation message.
Delete user /users/:id Allow users to delete their profile DELETE none a status code (201) if the user was deleted with a confirmation message
